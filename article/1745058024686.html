stable-diffusion-webui

  1. 安装python3.10，注意别安装高版本，点击webui-user.bat会提示安装合适的版本，卸载其他版本重新安装即可

  2. 设置git代理，设置国内pip加速. 注意使用阿里源，不要使用清华源, 会出现问题，参考文章：https://www.cnblogs.com/aligege/p/17481838.html

    pip config set global.index-url https://mirrors.aliyun.com/pypi/simple

  3. 修改webui-user.bat

    ## 显卡内存 8GB上
set COMMANDLINE_ARGS=--enable-insecure-extension-access
## 显卡内存 4G到8G
set COMMANDLINE_ARGS=--lowvram --opt-split-attention --enable-insecure-extension-access
## 显卡内存 小于等于4GB
set COMMANDLINE_ARGS=--medvram --opt-split-attention --enable-insecure-extension-access
## 无显卡，仅cpu运行
set COMMANDLINE_ARGS=--skip-torch-cuda-test --no-half --precision full --use-cpu all

  4. 点击 webui-user.bat

可能出现的问题以及解决方案：

  1. pip从github中安装错误

    RuntimeError: Couldn't install open_clip.
Command: "D:\Desktop\stable-diffusion-webui\venv\Scripts\python.exe" -m pip install https://github.com/mlfoundations/open_clip/archive/bb6e834e9c70d9c27d0dc3ecedeebeaeb1ffad6b.zip --prefer-binary
Error code: 2

    解决：

    手动下载 https://github.com/mlfoundations/open_clip/archive/bb6e834e9c70d9c27d0dc3ecedeebeaeb1ffad6b.zip 到 D:\Desktop\stable-diffusion-webui\venv\Scripts 目录，然后在该目录执行 

    .\python.exe -m pip instal bb6e834e9c70d9c27d0dc3ecedeebeaeb1ffad6b.zip

    即可安装，然后重新点击 webui-user.bat

  2. 模型下载慢的话，可以关闭重新打开，重新下载，也可以手动下载：

    Downloading: "https://huggingface.co/runwayml/stable-diffusion-v1-5/resolve/main/v1-5-pruned-emaonly.safetensors"
