  1. win10删除账户

    转载: https://zhuanlan.zhihu.com/p/345835874

    1. 打开注册表删除这两个

      HKEY_CURRENT_USER\Software\Microsoft\IdentityCRL\UserExtendedProperties\xxx@xx.com
HKEY_USERS.DEFAULT\Software\Microsoft\IdentityCRL\StoredIdentities\xxx@xx.com

    2. 注销，再登陆

    3. 然后继续删除两个IdentityCRL

      HKEY_CURRENT_USER\Software\Microsoft\IdentityCRL
HKEY_USERS.DEFAULT\Software\Microsoft\IdentityCRL

    4. 注销，再登陆。就可以在如下界面删除账户了

  2. win11菜单改回win10样式

    使用这个工具

    https://www.sordum.org/14479/windows-11-classic-context-menu-v1-1/

    或者

    https://zhuanlan.zhihu.com/p/524603591

    cmd执行，改为win10右键菜单

    reg add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve

    恢复为win11

    reg delete "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /va /f

  3. 获取win10锁屏壁纸

    C:\Users\A\AppData\Local\Packages\Microsoft.Windows.ContentDeliveryManager_cw5n1h2txyewy\LocalState\Assets

    文件夹下文件重命名添加 .jpg 后缀即可

  4. 软链接

    MKLINK [[/D] | [/H] | [/J]] Link Target
        /D      创建目录符号链接。默认为文件
            符号链接。
    /H      创建硬链接而非符号链接。
    /J      创建目录联接。
    Link    指定新的符号链接名称, 链接到
    Target  指定新链接引用的路径, 实际文件位置

    示例

    mklink /D C:\Users\Administrator\Desktop\vito\script\script E:\env\script
